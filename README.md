## Featured Projects

### FAIRS Roulette Player
Reinforcement learning (DQN) agent that learns to play roulette. Full-stack: PyTorch + FastAPI + React + Tauri desktop shell.
`Reinforcement Learning` `PyTorch` `React` `Tauri`

### XREPORT Radiological Reports
Vision-language model that generates descriptive radiology reports from chest X-ray scans.
`Computer Vision` `NLP` `Transformers` `Clinical AI`

### ADSMOD Adsorption Modeling
ML-driven adsorption isotherm fitting and prediction using NIST/ARPA-E databases.
`SciML` `Chemical Engineering` `NIST Databases`
